# Minerals Inventory

## Project Setup

To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/KimberlyHansen/Minerals.git
   cd Minerals
   ```

2. **Install dependencies**:
   Make sure you have Node.js and npm installed, then run:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```

   The application should now be running on [http://localhost:3000](http://localhost:3000).

## Features
- User authentication
- Inventory management for various minerals
- Search and filter functionality
- User-friendly web interface

## How to Use
1. Navigate to [http://localhost:3000](http://localhost:3000) in your web browser.
2. Sign in using your credentials or register a new account.
3. Once logged in, you can:
   - View the mineral inventory.
   - Add new minerals to your inventory.
   - Edit or delete existing entries.
   - Search for specific minerals using the search bar.

For further documentation or support, please refer to the project wiki or contact the maintainers.
